# KindaRandom

This software does some random things

## Getting Started

Clone, Fork, or Whatever.

### Prerequisites

Run the install.py to install all of the pip dependencies 

```
python install.py
```

Run kindaRandom.py and watch the magic. You should see a canvas and some other stuff.
